## 2. 고난이도 케이스 상세 분석 (Deep Dive)

### 2.1 False Positive (노이즈/오류 제어) ✨
> **검증 목표**: 검색 결과가 없거나 부적절한 요청에 대해 아무 상품도 추천하지 않아야 함.

| Query | 예상 의도 | Ground Truth | 비고 |
|:---|:---|:---|:---|
| "다이소에서 노트북 파나요?" | `미분류` (상품 없음) | `[]` (Empty) | 다이소 미취급 품목 |
| "명품 가방 있어요?" | `미분류` (상품 없음) | `[]` (Empty) | 브랜드 불일치 |
| "와인 셀러 추천해주세요" | `미분류` (상품 없음) | `[]` (Empty) | 미취급 품목 |
| "에어컨 설치해주세요" | `서비스` (상품 아님) | `[]` (Empty) | 서비스 요청 |

### 2.2 Hard Case (부정 조건 & 복합 추론) 🚀
> **검증 목표**: 단순 키워드 매칭으로는 오답이 나올 수밖에 없는 케이스를 LLM이 논리적으로 필터링하는지 검증.

1. **"락스 냄새 안 나는 화장실 곰팡이 제거제"**
   - **Scenario**: Negative Constraints
   - **Filters**: `[-락스, -냄새]`
   - **GT**: `[171, 151]` (곰팡이 제거 젤 등)
   - **검증 포인트**: '락스' 키워드가 들어간 락스 제품(오답)을 제외하는가?

2. **"플라스틱 없는 수납함"**
   - **Scenario**: Negative Constraints
   - **Filters**: `[-플라스틱]`
   - **GT**: `[93]` (종이/천 재질 수납함)
   - **검증 포인트**: 대부분의 수납함이 플라스틱인 상황에서 재질 정보를 파악하는가?

3. **"소독 스프레이 without 알코올"**
   - **Scenario**: Negative Constraints
   - **Filters**: `[-알코올]`
   - **GT**: `[295]`
   - **검증 포인트**: 영문 'without'과 화학 성분 제외 조건을 이해하는가?

4. **"강아지한테 해로운 성분 없는 간식"**
   - **Scenario**: Negative Constraints
   - **Filters**: `[-유해성분, 안전]`
   - **GT**: `[21, 37]`
   - **검증 포인트**: '해로운 성분'이라는 추상적 불만을 안전한 성분으로 치환하여 검색하는가?

5. **"냄새 안 나는 쓰레기 봉투"**
   - **Scenario**: Negative Constraints
   - **Filters**: `[-냄새, 탈취]`
   - **GT**: `[1, 557]`
   - **검증 포인트**: 쓰레기 봉투 자체의 냄새 혹은 냄새 차단 기능을 추론하는가?

6. **"플라스틱 아닌 천연 소재 도시락통"**
   - **Scenario**: Negative Constraints
   - **Filters**: `[-플라스틱, 천연소재]`
   - **GT**: `[127]`
   - **검증 포인트**: 밀폐용기 중 플라스틱이 아닌(유리/스텐 등) 제품을 선별하는가?

7. **"배터리 오래가는 충전식 건전지"**
   - **Scenario**: Complex Query
   - **Filters**: `[오래가는, 충전]`
   - **GT**: `[10, 12, 20]`
   - **검증 포인트**: 일회용 알카라인 건전지가 '충전식'이 아님을 구분하는가? (함정 문제)

### 2.3 Medium Case (암시적/주관적 니즈)
> **검증 목표**: 사용자의 모호한 요구사항을 구체적인 카테고리나 속성으로 매핑하는지 검증.

| Query | 니즈 분석 | GT Hints |
|:---|:---|:---|
| "곰팡이 제거" | 문제 해결(Problem Solving) → 청소용품 매핑 | `[171]` |
| "기름때 제거" | 문제 해결 → 주방 세제/수세미 매핑 | `[]` (관련 상품 탐색 필요) |
| "머리카락 막힘" | 문제 해결 → 배수구 뚫는 세제/도구 매핑 | `[76]` |
| "자취생 필수템 청소도구" | 페르소나(자취생) → 가성비, 다용도 | `[19, 547]` |
| "손이 안 트는 주방세제" | 고충 해결 → 저자극/순한 성분 | `[382, 407]` |
| "아기 피부에 안전한 섬유유연제" | 대상(아기) → 저자극/안전 | `[78]` |
| "가성비 좋은 물티슈" | 주관적(가성비) → 가격 필터 | `[39]` |
| "싸고 좋은 청소포" | 주관적(싸고 좋은) → 가격/평점 | `[547]` |
| "비건 주방세제" | 카테고리 중의성 → 생활용품 > 주방세제 | `[382]` |
| "민감성 피부 물티슈" | 카테고리 중의성 → 뷰티/위생 > 물티슈 | `[39]` |
